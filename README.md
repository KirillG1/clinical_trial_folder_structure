### Clinical trial project template

#### About the project
The current project is a template for clinical study data processing. 
It covers SDTM, ADAM, and TLF development with and without QC.

#### Project structure
- data [store corresponding datasets]
  - ADAM
  - SDTM
  - figure
  - tables
  - listings
  - qc_ADAM
  - qc_SDTM
  - qc_figure
  - qc_tables
  - qc_listings
  - raw
  - misc
- docs [study and supporting documents]
- code [supports main side]
  - adam
  - sdtm
  - figure
  - tables
  - listings
  - func
- qc_code [supports QC side]
  - adam
  - sdtm
  - figure
  - tables
  - listings
  - func
- report [stores TLFs]
  - figure
  - tables
  - listings
- result [stores main results]
